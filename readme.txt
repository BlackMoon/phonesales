Примечания по сборке/развертыванию.

1. В папке modules находится:
	 .ear - архив для deploy на Glassfish4. 
	отдельные  проекты - ejb, web

2. Для корректного отображения кирилических символов необходимо запускать Glassfish c JVM-параметром    
	-Dfile.encoding=UTF8

3. Если ejb разворачивается на отдельном сервере от веб-проекта, тогда необходимо пересобрать web-project, но поменять url-адреса к  веб-сервису.
(классы в пакете ~/org.bm.ui_*, поле Adress)

4. архив дампа базы MySQL - папка db/*.sql

5. файл-конфигурация соединения с базой данных - ~/*-ejb/Meta-inf/persistence.xml

6. Авторизация. В базе заведены 2 пользователя (c SHA-256 хешем):
	admin - пароль 123456
	user - пароль 123456

Редактировать данные может авторизованный пользователь. 
Редактировать админ. параметры (список пользователей) - пользователь с ролью admin.

7. Сборка/развертывание производится с помощью Maven командой clean package glassfish:deploy (clean package glassfish:deploy:redeploy). Pom.xml - файлы прилагаются. 

8. Проект на google - https://code.google.com/p/phonesales/
